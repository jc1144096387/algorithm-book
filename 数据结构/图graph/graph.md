# 图
图是一组由边连接的顶点。            
G = (V,E)           
V: 一组顶点
E: 一组边       
相邻顶点: 由一条边连接在一起的顶点          
度: 相邻顶点的数量

## 图基础（base.js）
- 图的表示
    + 邻接矩阵
    + 邻接表
    + 关联矩阵
- 创建Graph类(邻接表)
- 图的遍历
    + 广度优先搜索(Breadth First Search, BFS)
    + 深度优先搜索(Depth First Search, DFS)
- 拓扑排序
- 最短路径算法
    + Dijkstra算法
    + Floyd-Warshall算法
- 最小生成树
    + Prim算法
    + Kruskal算法
